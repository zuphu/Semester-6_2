My name is Anthony and I am in my final semester at Algonquin College studying Computer Engineering Technology - Computing Science. My final semester project consists of a web application that will be used by the RCMP to track drug trends in Ottawa. The application allows people to anonymously register and report information relating to drugs in specific zones. This application is being written in ASP.NET and is using SQL server.

In my spare time I experiment with game libraries. I have dabbled around with SDL and XNA. I created a game for a quarterly game jam called Ludumdare with a friend that you can see on my bitbucket page (Social link on the left and Additional Content). I have gained lots of practical coding experience while studying at Algonquin and look forward to putting my skills to use in the real world.
